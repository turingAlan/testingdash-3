
# AetherCore

Dashboard for onboading of sellers on ONDC


## Run Locally

Clone the project

```bash
  git clone 
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
```

Start the server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```


## Tech Stack

**Client:** Typescript, React, Next, TailwindCSS, Tanstack Query, Zustand
